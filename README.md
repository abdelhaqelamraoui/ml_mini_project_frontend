# Frontend du mini projet de Fin de Module en Maching Learning

Ce projet est une application frontend qui vise à fournir aux utilisateurs une interface utilisateur propre pour prédire les voitures marocaines. Il utilise des techniques d'apprentissage automatique pour effectuer des prédictions précises. Pour exécuter le projet, vous pouvez utiliser la commande suivante :

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
